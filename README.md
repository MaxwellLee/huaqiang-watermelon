# 华强买瓜 · 像素挑瓜指南

一个像素风的挑西瓜教学小游戏,灵感来自《征服》经典买瓜名场面。

## 🍉 在线试玩

**https://maxwelllee.github.io/huaqiang-watermelon/**

## 玩法

- 拖动旋转西瓜,凑近看 **纹路 / 瓜斑 / 瓜脐 / 瓜蒂**,华强自动点评
- 左键点瓜 **拍一拍** 听声音,点手掌 **掂一掂** 试分量
- 看、拍、掂三步走完,自动劈瓜揭晓:好瓜 / 生瓜蛋子 / 坏瓜 / 空心瓜
- 前四个瓜四种类型各出一个(顺序随机),右上角"开瓜图鉴"记录战绩,集齐四种有惊喜
- 外星语开场白 + 每种瓜专属揭晓配乐
- 每种瓜的外观、声音、分量都和真实挑瓜经验对应,玩完真能去挑瓜

## 本地运行

纯静态页面,无需构建,直接用浏览器打开 `index.html` 即可。

## 技术

- 场景 / 人物 / 道具:Canvas 代码手绘像素画
- 西瓜:Three.js 低分辨率渲染 + 最近邻纹理(复古 3D 像素瓜)
- 字体:[Zpix 像素字体](https://github.com/SolidZORO/zpix-pixel-font)
- 音效:Web Audio API 合成
